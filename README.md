# CONTROLE_FINANCEIRO
PROJETO_FINAL_AULA20 - SENAI

O objetivo do sistema é automatizar o controle financeiro pessoal, tornando a gestão de gastos mais rápida, organizada e confiável.
O problema identificado foi que a planilha criada no LibreOffice, apesar de funcional, apresentava instabilidade, lentidão e risco de perda de dados. 

Em resumo, o sistema transforma uma planilha instável em uma ferramenta robusta, automatizada e confiável, criada para melhorar o controle financeiro diário de forma simples e eficiente.
